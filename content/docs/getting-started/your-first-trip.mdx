**Your next trip starts here.** The app walks you through the rest; you only need the gist.

## Create the trip

From **Home**, **Trips**, or **Explore**, use the main **add** action (usually a **+** or **Add trip** entry). Give the trip a name you will still recognize months from now.

## Add a place

Search for where you are going, pick the result that matches (city and country help disambiguate), then set how long you are there. The trip needs at least one stop before it really feels alive on the map.

If you started with a name-only trip, open that trip and use the same idea—search, pick dates, save.

## After that

Open **Explore** to see the big picture, or stay on the trip to add notes, transport, and stays. If a pin looks slightly off on the basemap, zoom out once—your data is usually fine.

## Tips

- One real stop beats a pile of empty shells. Start small, then grow.
- On your phone, [install Nomu as a PWA](/getting-started/installing-nomu-pwa) so coming back feels like opening an app.
- You need to be **online** to create trips and add places.
