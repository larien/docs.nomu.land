Good bug reports save everyone time. You do not need to be technical—just specific.

## Include

- **What you expected** vs **what happened** (one sentence each is fine).
- **Device** (phone model or laptop OS) and **browser** if you are not using the installed PWA.
- **Rough time** (with timezone) so we can match logs.
- A **screenshot** if there is visible UI weirdness.

## Where to send it

Use in-app feedback or **[Contact](https://app.nomu.land/contact)** on [app.nomu.land](https://app.nomu.land). For **documentation** mistakes, use **Suggest an edit** at the bottom of any docs page to open a GitHub issue.
